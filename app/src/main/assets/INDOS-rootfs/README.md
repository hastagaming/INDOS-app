## 💠 Apa itu INDOS?
**INDOS** adalah sistem operasi kustom berbasis biner mandiri yang dirancang khusus untuk dijalankan di lingkungan **Termux** melalui `proot-distro`. INDOS tidak bergantung pada distro besar (seperti Alpine/Ubuntu), melainkan menggunakan pondasi biner pilihan yang ringan dan cepat.

### 🚀 Fitur Utama:
* **Super Lightweight:** Ukuran sangat kecil karena hanya berisi biner inti.
* **Neon Aesthetic:** Tampilan Banner Neon yang gagah setiap kali masuk.
* **Interactive Installer:** Menu instalasi yang mudah digunakan (dengan Tutorial).
* **One-Word Login:** Cukup ketik `indos` untuk mulai!

---

## 🛠️ Cara Instalasi
Pastikan kamu sudah menginstal `proot-distro` dan `git` di Termux sebelum memulai:

```bash
pkg install proot-distro git -y
```

### 🚀 Instalasi Cepat
Jalankan satu baris perintah ini di Termux:

```bash
git clone [https://github.com/hastagaming/INDOS](https://github.com/hastagaming/INDOS) && cd INDOS && ./install.sh
```
